7-Zip è utilizzabile tramite console. Per semplificarne l'utilizzo è sufficiente aggiungere il percorso dell'eseguibile alla PATH di sistema

## Archiviare

```
7z.exe a archivio.7z .\fileorigine.ext
```

## Estrarre file

```
7z.exe e fileorigine.zip
```
